# Chat App

## Description

Chat App - University Project

## Installation

```bash
$ cd api && npm install
$ cd web && npm install
```

## Running the backend

```bash
# start the app
$ make start

# stop the app
$ make stop

# destroy the app volumes
$ make destroy
```

## Web interfaces for dev env only

mongo-express: http://localhost:8081

minio: http://localhost:9000
